# ML-MV-for-Drones
Machine learning and vision for drones
Дрон снимает видео, кадры попадают на компьютер оператора, где ИИ распознаёт повреждения фасада и сохраняет отобранные фотографии вместе с координатами.
Библиотеки:
  GUI: Kiwi
  ML, CV: FastAI
